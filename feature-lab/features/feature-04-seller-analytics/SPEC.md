# Feature 04: Seller Analytics V2

## Status: ⏳ Planned

## Problem
Current analytics show basic page views and product views. Sellers can't see conversion funnels, revenue trends, or how they compare to category averages. Without actionable insights, sellers don't know what to improve.

## Solution
Enhanced analytics dashboard with:
- **Conversion funnel**: View → Cart → Order → Paid (with drop-off rates)
- **Revenue trends**: Daily/weekly/monthly revenue chart
- **Benchmarking**: "Your store ranks in top 20% for order conversion in Women's Clothing"
- **Product performance**: Best sellers, worst performers, opportunity products
- **Traffic sources**: WhatsApp referrals vs organic vs marketplace

## Acceptance Criteria
- [ ] Funnel visualization shows 4 stages with drop-off percentages
- [ ] Revenue chart with period switching (7d/30d/90d/custom)
- [ ] Category benchmarks show percentile ranking
- [ ] Product performance table sortable by views/orders/revenue
- [ ] Traffic source breakdown with percentages
- [ ] All charts mobile-responsive
- [ ] Data refreshes on page load (< 2s)
- [ ] Empty state for new sellers with 0 data
