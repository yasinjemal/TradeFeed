# Growth Strategy — 10,000 Sellers

**Date**: 2026-03-17  
**Goal**: Reach 10,000 active sellers within 12 months  
**Market**: South African WhatsApp-commerce small businesses

---

## 1. Current State

TradeFeed is in early-growth stage with a working product (marketplace, catalogs, orders, promotions, analytics). The platform needs to scale seller acquisition while maintaining quality.

### Growth Equation

```
Growth = (New Sellers × Activation Rate) − Churn
```

**Targets**:
- New sellers: 1,200/month by Month 6
- Activation rate (listed 3+ products): 60%
- Monthly churn: < 10%
- Active sellers = 10,000 by Month 12

## 3. Activation Strategy

Getting sellers to sign up is half the battle. Activation (listing 3+ products) is where value is created.

### Onboarding Funnel Optimization

```
Sign Up → Create Shop → List First Product → List 3 Products → First Sale
```

| Stage | Current Drop-off | Target | Strategy |
|-------|-----------------|--------|----------|
| Sign Up → Create Shop | ~70% | 85% | Simplify form, pre-fill from Clerk profile |
| Create Shop → First Product | ~50% | 70% | WhatsApp import CTA, AI product builder |
| First Product → 3 Products | ~40% | 60% | "List 3 products for free verification badge" |
| 3 Products → First Sale | ~20% | 35% | Share tools, marketplace promotion trial |

### Activation Tactics

1. **Day 0**: Welcome WhatsApp message: "Send us a product photo to create your first listing!"
2. **Day 1**: "You're 2 products away from a verified badge — send 2 more photos!"
3. **Day 3**: "Your store needs a description. Here's a template: [pre-filled]"
4. **Day 7**: "Share your store on WhatsApp — here's your link: [shortlink]"
5. **Day 14**: "Other sellers in [city] are getting X views/week — list more products to compete!"

### Gamification

- **Free Tier**: 10 products, basic features
- **Verified Badge**: 3+ products + complete profile → verification badge on marketplace
- **Pro Trial**: List 5+ products → auto-unlock 14-day Pro trial
- **Featured Shop**: Top 10% active sellers → featured in marketplace carousel

## 4. Referral Loops (Viral Mechanics)

### Loop 1: Seller-to-Seller Referral
```
Seller A → shares referral link in WhatsApp group
  → Seller B signs up → lists products
  → Seller A gets free Pro month
  → Seller A shares again (motivated by reward)
```

### Loop 2: Buyer-to-Seller Conversion
```
Buyer finds product on marketplace
  → Orders via WhatsApp
  → Receives: "You could sell here too! Create your free store"
  → Buyer becomes seller (many SA buyers also sell)
```

### Loop 3: Store Link Sharing
```
Seller shares store link on WhatsApp status
  → 200+ contacts see the link
  → Some contacts are also sellers
  → They discover TradeFeed through the store link
  → They sign up and create their own store
```

### Loop 4: Product Share Virality
```
Product shared via WhatsApp (formatted card with image/price)
  → Recipient opens link → browses marketplace
  → Sees "Create your own store — 2 minutes"
  → Signs up
```

### Loop 5: Market Group Effect
```
One seller in a WhatsApp group uses TradeFeed
  → Group members see professional product links
  → "How did you make that store?"
  → Multiple sellers from same group sign up
  → Network effect within communities
```

## 7. Viral Sharing of Store Links

### WhatsApp Status Sharing

Every seller gets tools to share professional product cards on WhatsApp Status:
- Auto-generated image card (product photo + price + store name)
- "Share to WhatsApp Status" one-tap button
- Formatted message: "🆕 Just listed! [Product] — R[price]. Shop my store: [link]"

### Link Preview Optimization

When a TradeFeed link is shared on WhatsApp, it shows:
- **Thumbnail**: Product image (high quality)
- **Title**: Product name + price
- **Description**: Short description + store name + city
- **CTA**: "View on TradeFeed"

This is already implemented via OG meta tags and `/api/og` dynamic images.

### Share Templates (Pre-filled Messages)

1. **New Product**: "🆕 Just added to my store! {product} — R{price}. See it here: {link}"
2. **Store Launch**: "🎉 My online store is live! Browse {count}+ products: {link}"
3. **Sale**: "🔥 SALE! {product} now R{sale_price} (was R{price}). Shop: {link}"
4. **New Drop**: "📦 New stock just dropped! {count} new items. Shop now: {link}"

### WhatsApp Group Engagement Tools

For sellers who sell in WhatsApp groups:
1. **Product catalog message**: Formatted list of all products with prices
2. **Order form link**: "Click to order: {link}" → opens pre-filled cart
3. **Stock update**: "🔄 Stock update: {product} — {quantity} left!"
